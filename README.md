# Student Performance Analyzer

This is a simple Python project developed to calculate a student's total marks, percentage, and grade.

## Description
The program takes marks of three subjects as input and calculates the total marks and percentage. Based on the percentage, it assigns a grade and gives a simple performance message.

## Features
- Calculates total marks
- Calculates percentage
- Assigns grade (A, B, C, Fail)
- Displays performance message

## How to Run
1. Download the Python file
2. Open it in any Python environment (IDLE/Colab)
3. Run the program
4. Enter the required inputs

- Python
